# AI Research OS 进化路线图

> 自进化的研究操作系统 — 从论文管理到智能研究伙伴

## 产品定位

**核心定位**：AI Research OS 是一个**自进化的研究操作系统**，能从使用中学习，持续改进。

**差异化价值**：不是静态工具，而是会"成长"的研究伙伴。

```

                              │
                              ▼

│   (解析、索引、检索、生成、可视化)                           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    进化引擎层                                │
├─────────────────────────────────────────────────────────────┤
│  Signal Collection  │  Gene Encoding  │  Evolution Engine  │
│  ─────────────────  │  ──────────────  │  ─────────────── │
│  • user_feedback   │  • successful_qa │  • EvoSkill      │
│  • query_success   │  • good_slides   │  • Evolver       │
│  • slide_quality   │  • research_pat  │  • Self-refine   │
│  • research_pat    │  • failed_att    │  • Gene accum    │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      记忆层                                 │
│   (Evolution Memory / Gene Pool / Capsule Store)            │
└─────────────────────────────────────────────────────────────┘
```

## 信号类型定义

| Signal | 触发条件 | 价值 |
|--------|----------|------|
| `user_feedback` | 用户纠正回答 | 学习正确模式 |
| `query_success` | 连续3次问答成功 | 强化检索策略 |
| `slide_quality` | PPT被多次查看 | 优化生成模板 |
| `research_pattern` | 相同领域查询聚集 | 发现研究热点 |
| `failure_avoid` | 失败后重试成功 | 规避错误模式 |
| `citation_discover` | 发现高价值引用 | 优化推荐算法 |

## Gene 设计

```yaml
Gene:
  name: "rag_success_pattern"
  trigger:
    - query_topic: "method_comparison"
    - paper_count: 2-5
  actions:
    - retrieval: "hybrid_bm25_embedding"
    - rerank: "cross_encoder"
    - generation: "cite_with_quote"
  outcome:
    success_rate: 0.75
    feedback_count: 12
  evolved: 3
```
